ASTRA – Python Voice Assistant 🤖

Astra is a lightweight Python-based voice assistant that works with voice commands or terminal commands.
It can open websites, search the internet, check weather, control system functions, and more.

---

✨ Features

- 🎤 Voice command recognition
- 🗣 Text-to-speech responses
- 🌐 Google search integration
- 📺 YouTube control
- ☁ Weather information
- ⚡ Internet speed test
- 🔋 Battery status
- 🧠 Memory system
- 💻 System commands (shutdown, restart, lock)
- ⌨ Terminal mode + Voice mode
- 🎯 Wake word detection ("Astra")

---

📂 Project Structure

astral_os/
│
├── main.py        # Main program loop
├── voice.py       # Speech recognition & speech engine
├── commands.py    # Command handler
├── system.py      # System operations
├── memory.py      # Astra memory storage
└── requirements.txt

---

⚙ Installation

Clone the repository:

git clone https://github.com/yourusername/astra-assistant.git
cd astra-assistant

Install dependencies:

pip install -r requirements.txt

Run Astra:

python main.py
